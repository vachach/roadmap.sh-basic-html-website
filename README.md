# Basic HTML Website

This project is a simple multi-page website built using only HTML as part of the [roadmap.sh Basic HTML Website project](https://roadmap.sh/projects/basic-html-website).

## About the Project

The goal of this project is to practice writing clean and semantic HTML without using CSS or JavaScript.  
It includes multiple pages with a consistent navigation structure and focuses on proper layout using only HTML.

## Pages

- Home
- Projects
- Articles
- Contact

## Features

- Semantic HTML structure
- Multi-page website
- Navigation bar shared across pages
- Contact form in HTML
- SEO-friendly meta tags
- Clean and simple HTML-only layout

## Technologies Used

- HTML5
